# RoboCar

Небольшой репозиторий-обёртка вокруг скетча для робокара.

Содержимое:
- [src/robocar/robocar.ino](src/robocar/robocar.ino) — основной Arduino-скетч (копия вашей исходной скетч-файла).
- [docs/USAGE.md](docs/USAGE.md) — краткая инструкция по загрузке скетча в Arduino IDE.

Как использовать:
1. Откройте Arduino IDE.
2. Откройте файл [src/robocar/robocar.ino](src/robocar/robocar.ino) как sketch.
3. Выберите плату и порт, затем загрузите.

Если нужно, могу добавить CI, плату по умолчанию или схему подключения.
