# Использование

1. Откройте Arduino IDE или VS Code с PlatformIO.
2. Откройте sketch: [src/robocar/robocar.ino](src/robocar/robocar.ino).
3. Выберите правильную плату (например, STM32, если используются порты PAx/PBx).
4. Подключите плату и загрузите скетч.

Примечание: Пины в скетче заданы как PA0, PA1 и т.д.; проверьте соответствие вашей плате.
